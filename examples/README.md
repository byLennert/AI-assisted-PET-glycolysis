# 📂 examples/ — 示例脚本说明

## 这个文件夹是做什么的？

`examples/` 文件夹包含用于帮助新手快速理解项目核心逻辑的**示例脚本**。
这些脚本使用**模拟数据**，不需要 API Key，可以直接运行。

---

## 📄 文件说明

| 文件名 | 用途 | 依赖 |
|--------|------|------|
| `quick_demo.py` | 贝叶斯优化完整流程演示（使用模拟数据） | 只需 `scikit-learn`, `scipy` |

---

## 🚀 如何运行

```bash
# 从项目根目录运行
python examples/quick_demo.py
```

---

## 📖 quick_demo.py 演示内容

1. **定义搜索空间**：5种还原剂 × 5种质子源 × 5种溶剂 = 125种组合
2. **用随机向量模拟嵌入**：无需 API，用随机数模拟化学特征向量
3. **初始实验**：手动选 5 个条件，测定产率（模拟）
4. **贝叶斯优化循环**：5 轮迭代，每轮推荐最有希望的实验
5. **结果分析**：仅用了 10/125（8%）的实验，找到了最优条件

**运行时间**：约 30 秒（含 GP 训练时间）

---

## 🔄 quick_demo.py 与真实项目的区别

| 方面 | quick_demo.py（演示） | llm_embedding_bo.py（真实） |
|------|----------------------|---------------------------|
| 嵌入向量 | 随机 4 维向量 | LLM 生成 1536 维向量 |
| 产率来源 | 数学函数模拟 | 实验室实际测量 |
| 搜索空间 | 5×5×5=125 | 10×10×12=1200+ |
| 运行时间 | 约30秒 | 首次约5分钟（生成嵌入） |
| API费用 | 免费 | 约 0.5-2 美元（首次） |

两者的**优化逻辑完全相同**，只是数据来源不同。
