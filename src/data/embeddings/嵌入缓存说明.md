# 嵌入向量缓存文件夹（Embeddings Cache）

## 这个文件夹是做什么的？

此文件夹存储 LLM（大语言模型）为每一种化学物质生成的**语义嵌入向量**，格式为 JSON 文件。

每次程序需要某化学物质的嵌入向量时，会：
1. **优先查找**此文件夹中是否已有缓存 → 有则直接读取（省钱省时）
2. 若无缓存 → 调用 OpenAI API 生成 → 保存到此文件夹

---

## 文件格式示例

每个文件名就是化学物质名称，内容如下：

**`AcOH.json`**（醋酸的嵌入向量）：
```json
{
  "molecule": "AcOH",
  "property": "Acetic acid (AcOH) is a weak organic acid with the chemical formula CH3COOH. It has a pKa of 4.76, making it a moderate proton donor. In Mg-mediated reductive coupling reactions, AcOH serves as a proton source that protonates organometallic intermediates after radical addition. Its moderate acidity avoids premature protonation of radical intermediates while ensuring efficient quenching of the metal-organic species...",
  "embedding": [0.0123, -0.0456, 0.0789, 0.0012, ...],
  "role": "proton_source"
}
```

**字段说明：**
| 字段 | 说明 |
|------|------|
| `molecule` | 化学物质名称 |
| `property` | LLM 生成的化学性质描述文本（供嵌入向量生成使用） |
| `embedding` | 1536维浮点数向量（text-embedding-3-small 模型输出） |
| `role` | 该物质的角色：`reductant`/`proton_source`/`solvent` |

---

## ⚠️ 注意事项

- **不要手动修改**这些 JSON 文件
- 如果某个文件损坏，直接删除该文件，程序下次运行会重新生成
- 第一次运行时此文件夹为空，程序会自动填充（需要网络和 API Key）
- 每次 API 调用大约消耗 **0.01–0.05 美元**，缓存后免费复用
