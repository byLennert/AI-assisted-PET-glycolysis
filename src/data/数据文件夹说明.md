# 📂 src/data/ — 数据文件夹说明

## 这个文件夹是做什么的？

`src/data/` 是项目的**核心数据目录**，存放所有候选化学品列表、初始实验数据以及 LLM 生成的嵌入向量缓存。

---

## 📄 文件清单与说明

| 文件/文件夹 | 说明 | 是否必须 |
|-------------|------|----------|
| `reductant.xlsx` | 候选**还原剂**列表（BO 的搜索空间之一） | ✅ 必须 |
| `proton_source.xlsx` | 候选**质子源**列表（BO 的搜索空间之一） | ✅ 必须 |
| `solvent.xlsx` | 候选**溶剂**列表（BO 的搜索空间之一） | ✅ 必须 |
| `init_experiments.xlsx` | **初始实验数据**，BO 从这里读取历史数据 | ✅ 必须 |
| `embeddings/` | LLM 生成的分子嵌入向量缓存（JSON格式）| 自动生成 |
| `all_possible_reductant.npy` | 还原剂嵌入向量矩阵缓存 | 自动生成 |
| `all_possible_proton_source.npy` | 质子源嵌入向量矩阵缓存 | 自动生成 |
| `all_possible_solvent.npy` | 溶剂嵌入向量矩阵缓存 | 自动生成 |
| `random_embedding_20.npy` | 随机投影矩阵（降维用） | 自动生成 |
| `uname_dict.json` | 文件名映射字典（特殊字符处理） | 自动生成 |

## 📊 各 Excel 文件格式

### `reductant.xlsx` — 还原剂候选列表
```
name                    | 描述
------------------------|-----------------------------------
Mg powder (325 mesh)    | 镁粉，325目，强还原剂 E0=-2.37V
Zn powder               | 锌粉，中等还原性 E0=-0.76V
...
```
> **重要**：`name` 列的名称必须与 `init_experiments.xlsx` 中的 `Reductant` 列完全对应！

### `proton_source.xlsx` — 质子源候选列表
```
name            | pKa  | 描述
----------------|------|----------------------------
AcOH            | 4.76 | 醋酸，基准质子源
Propionic acid  | 4.87 | 丙酸，pKa相近
...
```

### `solvent.xlsx` — 溶剂候选列表
```
name            | 类型       | 描述
----------------|------------|---------------------------
DMAc            | 极性非质子  | 基准溶剂（毒性高）
2-MeTHF         | 绿色醚类   | 生物基绿色替代
...
```

### `init_experiments.xlsx` — 初始实验数据
```
Reductant               | ProtonSource | Solvent | Yield | 备注
------------------------|--------------|---------|-------|----------
Mg powder (325 mesh)    | AcOH         | DMAc    | 85    | 基准条件
Mg powder (325 mesh)    | AcOH         | DMF     | 72    | 换溶剂
...
```
> **必须列**：`Reductant`, `ProtonSource`, `Solvent`, `Yield`（产率，0-100的数字）

## 📂 embeddings/ 子文件夹

存放 LLM 对每个化学物质名称生成的**语义嵌入向量**（JSON 格式），**第一次运行时由程序自动生成**，之后缓存以节省 API 调用费用。

单个嵌入文件示例 (`AcOH.json`):
```json
{
  "molecule": "AcOH",
  "property": "Acetic acid is a weak organic acid with pKa 4.76...",
  "embedding": [0.0123, -0.0456, 0.0789, ...],
  "role": "proton_source"
}
```

> **注意**：这些文件由程序自动生成，**不要手动编辑**。

---

## ✏️ 如何修改候选化学品列表？

如果你想添加新的还原剂/质子源/溶剂：
1. 直接在对应的 Excel 文件中添加新的一行
2. 程序下次运行时会自动调用 LLM API 生成新物质的嵌入向量
3. 嵌入向量会被缓存到 `embeddings/` 文件夹中

如果你想**缩减**候选范围以加快计算：
1. 删除 Excel 文件中不需要的行
2. 删除对应的 `.npy` 缓存文件（程序会重新生成）
